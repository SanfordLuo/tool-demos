# 用户注册

**请求URL**

- `/user-center/register`

**请求方式**

- POST

**请求参数**

|   名称   |   类型   |   是否必填   |   说明   |
|   ---   |   ---   |   ---   |   ---   |
| id | Int | 是 | id |

**请求示例**

```json
{
  "id": 111
}
```

**返回参数**

|   名称   |   类型   |   说明   |
|   ---   |   ---   |   ---   |
| id | Int | id |

**返回示例**

```json
{
  "code": 0,
  "message": "success",
  "data": {}
}
```
